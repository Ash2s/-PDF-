# PDF 表格自动填写工具

AI驱动的PDF表格自动填写工具，上传学生简历+PDF申请表，自动识别字段并填充。

## 功能特点

- **AI自动识别**：自动识别PDF表单中的填写字段
- **智能填充**：根据学生简历自动填写姓名、学校、家庭信息等
- **多表单支持**：一次处理多个PDF申请表，Tab切换预览
- **在线编辑**：可预览PDF、修改字段值、添加新字段
- **下载填写版**：生成填写好的PDF，支持单个或批量下载

## 快速开始

### 方式一：本地运行

**1. 安装依赖**
```bash
pip install -r requirements.txt
```

**2. 配置API密钥**
```bash
# 复制配置文件
cp .env.example .env

# 编辑 .env 文件，填入你的 MIMO_API_KEY
MIMO_API_KEY=你的密钥
```

**3. 启动服务**
```bash
python app.py
```

**4. 访问**
打开浏览器访问 http://localhost:8001

### 方式二：Docker运行

```bash
# 构建镜像
docker build -t pdf-form-filler .

# 运行容器
docker run -p 8001:8001 -e MIMO_API_KEY=你的密钥 pdf-form-filler
```

## 使用流程

1. **上传学生资料**：支持 PDF、Word、图片格式
2. **上传PDF申请表**：支持多个PDF同时上传
3. **点击"开始处理"**：AI自动识别字段并填充
4. **预览和修改**：
   - 左右箭头切换PDF页面
   - 在"修改意见"框输入修改指令
   - 点击"重新生成"应用修改
5. **下载结果**：
   - "下载填写版"：下载当前表单
   - "下载全部"：下载所有填写好的表单

## 修改指令示例

在修改意见框中输入自然语言指令：

```
把学校名称改成"香港大学"
```
```
中文成绩填A
```
```
父亲职业改成"工程师"
```
```
添加国籍字段，填写"中國"
```

## 环境要求

- Python 3.10+
- MIMO API 密钥（用于AI识别）

## 获取MIMO API密钥

1. 访问 [MIMO API](https://api.xiaomimimo.com)
2. 注册账号并申请API密钥
3. 将密钥填入 `.env` 文件

## 常见问题

**Q: 为什么识别不到字段？**
A: 确保PDF是可读的（非纯扫描件），且MIMO API密钥正确配置。

**Q: 家庭信息只填写了父亲，母亲没有？**
A: 已在最新版本修复，确保PDF表格中有明确的(1)(2)编号区域。

**Q: 如何填写国籍信息？**
A: 国籍默认填写"中國"（繁体中文），如需修改请在修改意见中说明。

**Q: 预览只能看到第一页？**
A: 确保使用最新版本，已修复页数显示问题。
